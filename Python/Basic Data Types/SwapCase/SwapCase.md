## 🔁 Problem: Swap Case

### Description
You are given a string and your task is to **swap the cases** of all characters:
- Convert all **lowercase letters** to **uppercase**
- Convert all **uppercase letters** to **lowercase**

---

### 🧪 Examples

#### Sample Input
Www.HackerRank.com

#### Sample Output
wWW.hACKERrANK.COM
